先用cd 進入folder
打 git init 創建.git資料夾
git add . 將工作目錄改變資料推上暫存區
git commit -m "xx" 將暫存區資料推進儲存庫

git branch xxx  xxx為分支名稱
git merge xxx 可將現在分支與xxx分支合併,若在與一新分支合併需手動解決衝突


git push -u origin xxx 可推分支上github

merge要領
